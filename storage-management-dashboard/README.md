# Storage Management Dashboard

## Overview
The Storage Management Dashboard is a web application designed to help users manage and monitor their storage inventory efficiently. It provides a user-friendly interface for viewing storage metrics, managing inventory items, and configuring application settings.

## Features
- **Storage Overview**: Displays key metrics and statistics related to storage usage.
- **Usage Chart**: Visualizes usage data in an interactive chart format.
- **Inventory Management**: Allows users to view, edit, and manage inventory items.
- **Reusable Modal Component**: Provides a consistent way to display information and prompts to users.
- **Home and Settings Pages**: Integrates various components and allows users to configure application settings.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (version 6 or higher)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd storage-management-dashboard
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm start
```
The application will be available at `http://localhost:3000`.

### Running Tests
To run the tests for both client and server, use:
```
npm test
```

## Folder Structure
- `src/client`: Contains the React application code.
- `src/server`: Contains the backend server code.
- `tests`: Contains unit tests for both client and server components.
- `package.json`: Configuration file for npm.
- `tsconfig.json`: TypeScript configuration file.
- `.eslintrc.js`: ESLint configuration file.
